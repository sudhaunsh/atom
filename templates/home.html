Welcome to the Atom Framework

Atom Framework is a web framework designed to access Atom Ai's services. It is designed to be simple and easy to use. It is also designed to be easily customizable and extendable. It is built on top of the Flask framework and uses Jinja2 templating.

You can access various services provided by Atom AI such as:

  • Atom AI
  • DenV
  • Ruru electrochemical analysis
  • TeztItAll
However, to access these services, you need a licensed copy of the respective software. Atom framework acts as a back end processing unit to aid in cloud computing, gathering and processing data. Some services like Ruru electrochemical analysis is currently under development and will be available soon. please find more information about all the services below. To access the services simply open the app, or go to the services page for more information.

Services provided by Atom AI

Atom AI

Atom AI is a machine learning and artificial intelligence software. It is designed to be easy to use and easy to understand. It is also designed to be easily customizable and extendable. It is built on top of the Tensorflow framework and uses Keras for deep learning. It is also built on top of the Flask framework and uses Jinja2 templating. It is designed to be used with the Atom Framework. It is currently under development and will be available soon. The main purpose of Atom AI is to provide Doctors, Engineers, Scientists and Researchers with a tool to aid in their research and work, and day to day life.
Atom AI can help with tasks like:

  • Simple and complex calculations
  • Data analysis
  • Data visualization
  • Electrochemical analysis and information
  • Medical diagnosis
  • Medical image analysis
  • Medical data analysis
  • etc

DenV

DenV is a software designed to aid in the post-diagnostic care of Dengue fever. It is designed to be easy to use and easy to understand. It is also designed to be easily customizable and extendable. It is built on top of the numpy and pandas frameworks and uses Keras for deep learning. It also uses CustomTKinter framework for the GUI. Doctors can download a prelicensed copy of DenV from DenV and the documentation and installation guides.
DenV can help with tasks like:

  • Prediction of the severity of Dengue fever
  • Prediction of the duration of Dengue fever
  • Prediction of the recovery time of Dengue fever
  • Prediction of the treatment required for Dengue fever
  • Prediction of the platelet count
  • Confidence of the report
  • Necessary precautions like hospitalisation
  • etc
